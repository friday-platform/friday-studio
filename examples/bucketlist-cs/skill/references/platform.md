# Platform Context

Bucketlist Rewards is a SaaS employee recognition platform.

## URLs
- Platform: companyname.bucketlist.org or companyname.bucketlistrewards.ca
- Help Center: support.bucketlistrewards.com
- Support email: support@bucketlistrewards.com

## Features
- Recognitions and awards (peer-to-peer and manager-driven)
- Points system: "points to give" (allotted by role, expire on refresh) and
  "earned points" (redeemable, do not expire by default)
- Rewards marketplace: gift cards, experiences, charity donations, company items
- Integrations: Slack, Microsoft Teams, HRIS (ADP, BambooHR, HiBob)
- SSO: Google, Okta, ADFS, Azure AD / Entra ID
- Leaderboards and Insights dashboards (admin-only)
- Widgets for embedding recognition feeds

## Roles
- Platform administrators / company admins: manage settings, users,
  integrations, awards, reports
- Managers: approve awards, view team reports
- End users / employees / team members: send recognitions, redeem rewards,
  manage profiles

## Common support categories (by volume)
- Redemption issues: gift card not received, vendor problems, expedite requests
- Email/name change requests
- Password reset / login issues
- Points balance: low funds, missing points, point refunds
- Platform configuration: awards, roles, notifications, groups
- SSO setup and troubleshooting
- HRIS integration setup
- Reporting requests
- Amazon returns
- Company-specific marketplace items (physical shipping)
