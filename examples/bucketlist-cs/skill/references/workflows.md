## Common resolution actions
- Resend gift card / invitation email
- Process physical item order (ships in 2-2.5 weeks)
- Refund / return points
- Update user account (email, name, role, group)
- Send password reset link
- Fund platform rewards float
- Remove / deactivate user
- Generate and send report
- Escalate to redemptions or technical team
- Redirect to customer's internal HR/IT/admin team
- Log feature request

## Information agents commonly need from customers
- Screenshot of the issue
- Company name
- Redemption/order ID
- Email address on the account
- Shipping address (for physical items)
- Size/color preference (for branded merchandise)
- Alternate email address (when primary doesn't receive emails)

## Timelines
- Gift card (automated): same day
- Gift card (manual): 1-3 business days
- Physical item shipping: 2-2.5 weeks
- Support response: 1-2 business days
- Hours: Monday to Friday, 9am to 5pm EST
